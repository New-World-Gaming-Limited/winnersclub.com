此计算器的工作原理

独赢及名次投注是两笔本金相同的独立投注：一笔投注所选对象获胜 (独赢)，另一笔投注所选对象进入名次 (在特定名次范围内完赛)。总成本是您单位本金的两倍。

独赢部分仅在所选对象获得第一名时支付。回报为：独赢本金 x 独赢小数赔率。

名次部分在所选对象进入支付名次范围内时支付。名次赔率根据独赢赔率计算：名次赔率 = ((独赢赔率 - 1) x 分数) + 1。例如，对于赔率为 8.00 且分数为 1/4 的马匹，名次赔率 = ((8.00 - 1) x 0.25) + 1 = 2.75。

名次分数和支付的名次数根据比赛类型和参赛规模而异。赛马的标准规则通常为：5-7 匹参赛马通常支付前 2 名的 1/4；8 匹以上支付前 3 名的 1/4；16 匹以上的让磅赛支付前 4 名的 1/4；一些大型让磅赛会支付前 5-6 名的 1/5。

如果所选对象获胜，独赢和名次部分都会支付。如果它获得第 2 或第 3 名 (在名次范围内)，则仅支付名次部分。如果它未进入支付名次，则两部分都输掉，您将损失全部双倍本金。

对于赔率较高 (通常小数赔率超过 8.00) 的马匹，独赢及名次投注特别有吸引力，因为即使独赢部分输掉，名次部分仍能带来利润。对于赔率低于 3.00 的热门选择，名次回报通常不足以支撑双倍本金的支出。

常见问题

标准赛事支付多少个名次？

这取决于参赛规模和比赛类型。5-7 匹参赛马通常支付 2 个名次；8 匹以上支付 3 个名次；16 匹以上的让磅赛通常支付 4 个名次。您所选博彩公司的具体条款将覆盖这些默认设置。

名次分数是什么意思？

名次分数会降低独赢赔率以计算出名次赔率。9.00 独赢赔率的 1/4 分数得出的名次赔率为 ((9-1) x 0.25) + 1 = 3.00。如果您的马匹进入名次，名次部分将按此价格赔付。

独赢及名次投注划算吗？

对于竞争激烈赛事中赔率较高的马匹，独赢及名次投注非常划算。对于赔率较低的热门马，微薄的名次收益很难抵消双倍本金的投入。

我可以进行走地独赢及名次投注吗？

包括 Stake 在内的大多数博彩公司仅在特定赛事中限制走地独赢及名次投注。赛前独赢及名次市场始终适用于列出的赛事。

如果比赛因退赛而缩减规模会怎样？

如果参赛马匹数量降至阈值以下 (例如从 8 匹降至 5 匹)，博彩公司可能会减少支付名次数或更改分数。请务必在比赛开始前查看更新后的条款。
