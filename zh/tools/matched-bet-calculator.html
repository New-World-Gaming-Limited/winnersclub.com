此计算器的工作原理

匹配投注是一种技术，通过在博彩公司进行反向投注（Back Bet）并在博彩交易平台进行对赌投注（Lay Bet）来覆盖市场的两种结果，从而消除损失风险并从免费投注和奖金中提取价值。

最佳对赌本金的计算旨在使两种情况下的利润相等。对于资格投注（真钱），公式为：对赌本金 = (反向本金 x 反向赔率) / (对赌赔率 - 佣金率)。对于不退还本金的免费投注：对赌本金 = (反向本金 x (反向赔率 - 1)) / (对赌赔率 - 佣金率)。

对赌责任是您必须在交易账户中持有的用于支付对赌投注的金额：责任 = 对赌本金 x (对赌赔率 - 1)。这并非损失的资金，如果对赌获胜，它将退还。

对于资格投注，您通常会产生少量损失，因为反向赔率和对赌赔率很少完全一致，且交易佣金适用。这种资格损失是解锁免费投注的成本。

对于免费投注（不退还本金，SNR），计算会有所不同，因为您在反向端没有冒自己的本金风险。免费投注的净利润通常为面值的 70-90%，具体取决于反向和对赌赔率的匹配程度以及交易佣金水平。在 5% 佣金且赔率接近的情况下，20 个单位的免费投注通常可以提取 14-17 个单位的保证利润。

交易佣金会降低有效对赌赔率。在 3.10 的对赌中收取 5% 的佣金意味着实际对赌赔率为 3.10 - (3.10 - 1) x 0.05 = 3.10 - 0.105 = 2.995。计算器会自动将佣金应用于所有对赌获胜情况。

更接近的反向和对赌赔率会产生更小的资格损失。始终寻找与反向价格相差 0.10.0.15 十进制以内的对赌赔率，以保持资格损失在可控范围内。

常见问题

什么是 SNR 免费投注？

SNR 代表“不退还本金”。当免费投注获胜时，您仅收到奖金，而不包括原始本金金额。这是包括 Stake 在内的博彩公司提供的最常见免费投注类型。

我应该使用多少交易佣金率？

主要交易平台通常对每个市场的净利润收取 2-5% 的佣金。检查您的特定交易账户，高级会员有时支付更低的费率。此计算器中的默认值为 5%。

为什么我有时会在资格投注中产生亏损？

当反向和对赌赔率存在差异时，就会产生资格损失。对赌赔率通常略高，因为交易平台收取佣金，从而产生了差距。这笔小额损失是获得免费投注或奖金资格的成本。

如何找到最佳对赌赔率？

比较主要博彩交易平台的赔率。流动性较高的市场（例如顶级足球联赛）通常具有更窄的反向对赌价差，从而减少资格损失。

匹配投注是无风险的吗？

当正确执行匹配的反向和对赌投注时，匹配投注本身在投注结果方面的财务风险极小。主要风险在于操作失误（输入错误的本金或赔率）、博彩公司奖金条款变更以及被识别为匹配投注者的账户限制。
