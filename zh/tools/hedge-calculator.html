此计算器的工作原理

对冲投注是指在您的原始头寸建立后（通常是原始赔率已向有利于您的方向移动时），在相反的结果上进行投注。目标是无论发生哪种结果，都能保证获利或减少最大损失。

对于 "锁定等额利润" 模式，计算器会找出使两种情况下的净利润相等的对冲金额。公式为：对冲金额 = (原始投注额 x 原始赔率) / 对冲赔率。

具体来说：在 4.00 赔率下投入 50 个单位的原始投注，获胜可返还 200（利润 150）。要在 1.40 赔率下对冲，所需投注额 = 200 / 1.40 = 142.86。如果原始投注获胜：利润 = 200 - 50 - 142.86 = 7.14。如果对冲投注获胜：利润 = 142.86 x 1.40 - 50 - 142.86 = 7.14。两种情况下的利润相同。

"保证无损失" 模式计算所需的对冲金额，以便任何结果都不会导致损失。这就是盈亏平衡对冲。在上述示例中，任何能覆盖 50 单位原始投资的对冲金额都符合条件。

"自定义金额" 模式允许您输入任何对冲金额，并并排查看产生的利润或亏损情况。当您只想进行部分对冲而不是完全锁定结果时，这非常有用。

请注意，对冲总是以减少最大收益来换取确定性。合适的对冲金额取决于您的风险承受能力以及自原始投注以来盘中价格的变动幅度。

常见问题

我应该在什么时候进行对冲？

当您的原始选择已经明显向有利于您的方向移动，并且您希望在赛事结束前锁定利润时，请进行对冲。常见情况包括足球队在半场结束时领先，而您希望无论下半场结果如何都能保证回报。

对冲会减少我的利润吗？

是的。通过在两个结果上都下注，您投入了额外的资金。锁定的利润将始终低于您仅凭原始投注可能获得的最大潜在收益。

我可以对冲过关投注吗？

是的。如果过关投注的最后一项待定，您可以对该项的相反结果进行对冲。将原始过关投注的潜在派彩作为您的 "原始投注额"，并将最后一项的赔率作为对冲赔率。

什么是部分对冲？

部分对冲是指在相反结果上投入比完全锁定所需的金额更少的赌注。它在不完全锁定利润的情况下减少了风险敞口，从而为原始投注留出了一些上涨空间。

对冲和套利是一样的吗？

不同。套利是在赛事开始前，同时从不同的博彩商处对两个结果进行投注。对冲是在第一笔投注已经完成后进行的第二笔投注，通常是在头寸已经向有利于您的方向移动时。
