此计算器的工作原理

套利（也称为 arb）发生在某个市场中所有结果的合并隐含概率低于 100% 时。通过根据每个结果的隐含概率比例分配注金，您可以保证无论结果如何都能获得利润。

对于二路市场：套利% = (1/o1) + (1/o2)。对于三路市场：套利% = (1/o1) + (1/o2) + (1/o3)。如果套利% 小于 1.00，则存在套利机会。

每个结果的注金分配：注金_n = (总额 x (1/赔率_n)) / 套利%。这确保了每个结果支付的总金额相同：总额 / 套利%。保证利润为 (总额 / 套利%) - 总额。

例如：o1 = 2.20, o2 = 2.10, o3 = 3.60。套利% = 0.4545 + 0.4762 + 0.2778 = 1.2085。这高于 1.00，因此不存在套利。但如果 o3 为 5.00，套利% = 0.4545 + 0.4762 + 0.2000 = 1.1307，仍然没有套利。对于三路市场的套利，通常需要所有三个结果同时被赋予慷慨的定价。

寻找套利机会需要同时比较多家博彩公司的价格。实时差异并不常见且存在时间很短，因为博彩公司会迅速调整。最常见的真实套利机会产生于促销赔率提升或校正缓慢的错误赔率。

计算器不扣除佣金。如果其中一注是在博彩交易所进行的，则必须计入对冲佣金（通常为 2-5%），以确定净套利是否仍然成立。有关考虑佣金的计算，请参阅配对投注计算器。

常见问题

真正的套利机会多久出现一次？

信誉良好的博彩公司之间真正的套利机会非常罕见，一旦出现几分钟内就会消失。在多家博彩公司竞争且错误或赔率提升产生临时差异的大型赛事期间，它们更为常见。

什么样的最低套利百分比才值得进行套利投注？

任何低于 100%（十进制形式小于 1.00）的套利百分比在技术上都是盈利的。然而，在考虑到最小注金取整和时间成本后，大多数套利者追求至少 1-2% 的保证利润率。

我需要在多家博彩公司开设账户吗？

是的。套利产生于不同博彩公司之间的价格差异。在标准市场中，单一博彩公司无法针对自身提供套利机会。

我可以将此用于滚球盘市场吗？

可以，但滚球盘市场变化非常快。您必须能够几乎同时下注。下注之间的价格变化可能会消除套利甚至导致亏损。

套利投注合法吗？

套利投注是合法的，但一些博彩公司会限制被识别为套利者的账户。请务必检查条款和条件，并注意潜在的注金限制。
