Sugar Rush 1000 hoạt động như thế nào

Sugar Rush 1000 runs on a 7x7 grid using cluster pays rather than paylines. A cluster is 5 or more matching symbols touching horizontally or vertically. When a cluster forms, the involved symbols are removed and new ones cascade from above in the tumble mechanic. Consecutive cascades within one spin continue until no new cluster forms.

The name references the 1000x tối thiểu vòng quay miễn phí multiplier mechanic, which distinguishes this from the original Sugar Rush. The grid is filled with candy symbols of varying values. Low-pay symbols include pink and yellow candies; premium symbols include lollipops, gummy bears, and the chocolate bar. The chocolate bar is the highest-paying symbol.

Multiplier symbols appear on the grid and function as modifiers. These apply to any cluster win that touches them. In the base game, multipliers increase the win value for individual cluster hits. The 7x7 grid allows for more complex cluster formations than a 6x5 grid, creating more tumble cascade opportunities.

Scatter symbols (the wrapped candy) trigger the vòng quay miễn phí feature. Very high volatility means the base game frequently delivers minimal cluster wins over extended spin sequences. The 7x7 grid and cluster mechanic create large symbol sets, spreading probability across more positions.
