受け取り方法

登録、認証、入金の後、ライブサポートで MAXBET を告げるだけ。ボーナスはオペレーター発行です。

1. 扉を開く

アフィリエイトリンク経由で Stake.com へアクセスし、「登録」をクリック。メール、ユーザー名、パスワード、生年月日を入力します。

2. Level 3 まで認証

Stake はウェルカムボーナス支払い前に KYC Level 3 を要求します。身分証明書、住所証明、プラットフォームが要求する Level 3 追加書類をアップロードしてください。ウェルカムオファーはこの審査の通過が条件です。

3. 初回入金（最低$10）

$10〜$1,500の範囲で入金します。200%マッチは$3,000ボーナス上限まで線形にスケールします。$1,500以上の入金は通常通り処理されますが、ボーナスの増加は止まります。

4. ライブチャットを開き、MAXBET をリクエスト

入金が完了したら、Stake のライブサポートにメッセージを送り、コード MAXBET で登録したこととウェルカムボーナスのクレジットを希望する旨を伝えてください。サポートが資格確認後、24〜48時間以内に200%マッチがアカウントに反映されます。

重要: Stake.com のウェルカムボーナスは自動適用フィールドではなく、サポート発行のクレジットです。アフィリエイトリンク経由で登録し、KYC Level 3 を完了し、入金後にライブチャットを開いて MAXBET を告げてください。（入金額+ボーナス）× 40 ウェジャリング要件のロールオーバー進捗は、有効化後に VIP タブで確認できます。

ウェジャリング: 入金+ボーナス合算40倍

プレイスルーはボーナスだけでなく、入金額とボーナスの合算に適用されます。$500入金、$1,000ボーナス受取の場合、$1,500 × 40 = $60,000 のウェジャリングが必要です。上限を狙う前に計画を立ててください。

KYC Level 3 必須

Stake は段階的な認証を採用しています。Level 1 でベット可能、Level 2 で入金上限が上がり、Level 3 はウェルカムボーナスと高額出金を解放する書類審査レベルです。設定 → 認証から身分証明書、住所証明、資金源証明書類を提出します。

ゲームタイプ別の貢献率

ハウスエッジ4%以上のカジノスロット・ゲームはウェジャリングに100%貢献。スポーツベットは75%貢献。ライブディーラーゲームと低 RTP スロット（ハウスエッジ4%未満）は貢献率が低下または0になります。Stake Originals か閾値4%以上のスロットを中心にプレイしましょう。

クレジットの反映方法

初回入金後、ライブサポートに連絡して MAXBET を告げてください。KYC Level 3 ステータスを確認後、24〜48時間以内に200%ボーナスがクレジットされます。自動適用の紹介フィールドはなく、ウェルカムボーナスはオペレーターが手動で発行します。

新規ユーザー・初回入金のみ

MAXBET は Stake.com の新規アカウントの最初の対象入金に適用されます。既存アカウントは遡って申請できません。このウェルカムオファーでは、2回目以降の入金はマッチされません。

最低入金 & 制限国

オファーを有効にする最低入金は$10、最大対象入金は$1,500。米国、英国（2025年3月以降）、オーストラリア、欧州大陸の大部分は stake.com では利用不可。米国のプレイヤーには専用のStake.us ウェルカム、その他の方はミラーページで地域ドメインを確認してください。

典型的な価値は1コードあたり$1〜$5で、チャンネルを通じて毎日複数投稿されます。積み重なります。Gold ティア以上の VIP プレイヤーは、メールや限定 Telegram サブチャンネル経由で額面の高いプライベートコードも受け取れますが、それらは公開されることはありません。
