ポーカー FAQ

Stakeポーカーのソフトウェアは何ですか？

Stakeポーカーは独自開発のウェブブラウザベースプラットフォーム。ダウンロード不要でChromeやSafariからアクセス可能。モバイルブラウザにも完全対応している。

HUDや追跡ソフトは使用できますか？

ハンド履歴のダウンロードは可能。HUDについてはStake利用規約を確認すること。リアルタイムHUDの可否はプラットフォームポリシーによる。

ボーナスはポーカーにも適用されますか？

ウェルカムボーナス（コード MAX3000）はポーカーでの賭けにも対応。ただしロールオーバー条件のカウント率がスポーツ・カジノとは異なる場合がある。詳細はStakeプロモページで確認。

バッドビートジャックポットの発動確率は？

発動には極めて強いハンド（クワッズ以上）で負けることが必要なため、発動確率は低い。累積額は公開されており、発動前は常に$1M保証が維持される。

Run It Twiceはいつ使えますか？

両プレイヤーがオールインの状態でボードカードが残っている場合に選択可能。両者の同意が必要。ショートハンドテーブルで最も活用されている機能。
