When a cluster forms, the involved symbols are removed and new ones cascade from above in the tumble mechanic. Consecutive cascades within one spin continue until no new cluster forms.

The name references the 1000x minimum اسپین رایگان multiplier mechanic, which distinguishes this from the original Sugar Rush. The grid is filled with candy symbols of varying values. Multiplier symbols appear on the grid and function as modifiers. These apply to any cluster win that touches them. In the base game, multipliers increase the win value for individual cluster hits.
