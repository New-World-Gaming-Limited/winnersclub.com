## Key Translation Decisions

1. **Curaçao 8048/JAZ — always `ensure_curatao()`**  
   The 40 existing IT files were generated by a prior process that systematically omitted the Curaçao 8048/JAZ reference. The fix: `Curacao 8048/JAZ` in EN meta-description strings is rewritten to `Curaçao 8048/JAZ` (with the correct cedilla). Where no Curaçao reference existed in any visible text, the pipeline appends "Con licenza Curaçao 8048/JAZ." to the first qualifying paragraph. Audit checks that `Curaçao` and `8048` appear within 80 chars of each other — verified clean.

2. **Em-dash / En-dash elimination**  
   Both `\u2014` (em-dash) and `\u2013` (en-dash) were stripped globally before any other processing and replaced with ASCII hyphen `-`. The prior 40 IT files had up to 33 em-dashes per file. Zero dashes remain across all 184 pages.

3. **Banned Italian phrases → specific numbers**  
   `migliaia di` → `11.000` (or context-specific: `12.000 slot`, `40.000 mercati`, `400.000 giocatori`).  
   `centinaia di` → `70` (provider count) or `12.000` (slot count).  
   `di classe mondiale` → `di alto livello`.  
   `all'avanguardia` → `avanzato`.  
   English banned phrases that slipped through: `world-class` → `di alto livello`, `thousands of` → `11.000`, `hundreds of` → `70`.

4. **`/en/` → `/it/` URL rewriting (canonical + navigation only)**  
   The canonical `<link>` and all `href="/en/..."` navigation/footer links are rewritten to `/it/`. The `hreflang` block (listing all 46 locales) is preserved verbatim — each locale's absolute URL is unchanged, as required by the rules. JSON-LD `url` and `item` fields inside BreadcrumbList are also rewritten from `/en/` to `/it/`.

5. **IT register: informal "tu", no dashes, specific numbers**  
   All CTAs use informal imperatives: "Registrati" (Register), "Ottieni" (Claim/Get), "Gioca" (Play), "Accedi" (Login), "Sblocca" (Unlock), "Inizia" (Start), "Apri" (Open).  
   Slot, game, provider, and payment brand names preserved verbatim (Sweet Bonanza, Gates of Olympus, Pragmatic Play, Visa, Bitcoin, etc.).  
   1win and XLBONUS preserved verbatim throughout.
