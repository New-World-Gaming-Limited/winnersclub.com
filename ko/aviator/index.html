게임 내 실시간 베팅 피드에는 다른 플레이어의 베팅금, 캐시아웃 목표 또는 "비행 중" 상태, 당첨금이 실시간으로 표시됩니다. 화면에는 실시간 배수 히스토리가 스크롤됩니다. 이는 과거 결과를 보여줄 뿐 미래를 예측하지 않습니다. 라이브 채팅이 피드 옆에서 실행되며, Rain 프로모는 무작위 간격으로 활성 채터에게 무료 베팅을 지급합니다.

SHA-512 검증 가능한 공정성

베팅 후 결과가 조작되지 않았음을 확인하는 방법.

Aviator의 공정성 메커니즘은 SHA-512 해시 체인을 사용합니다. 각 라운드의 크래시 포인트는 다음을 결합하여 생성됩니다:

  1. Spribe의 서버 시드
  2. 해당 라운드에서 베팅을 건 첫 세 명 플레이어의 시드
  3. SHA-512 해시로 결합

Spribe는 다음 1,000만 라운드의 해시를 사전에 공개합니다. 각 라운드 종료 후 서버 시드가 공개되어 결과를 독립적으로 검증할 수 있습니다. 플레이어 시드가 해시에 포함되므로 Spribe나 Stake 어느 쪽도 단독으로 결과를 미리 결정할 수 없습니다. 실제 플레이어가 베팅을 걸기 전까지 크래시 포인트는 알 수 없습니다. 게임 내 검증 가능한 공정성(Provably Fair) 탭에서 플레이한 모든 라운드의 서버 시드, 결합 해시, 결과를 확인할 수 있습니다.
